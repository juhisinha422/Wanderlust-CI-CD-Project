𝗧𝗘𝗖𝗛 𝗦𝗧𝗔𝗖𝗞
• Jenkins — CI/CD Automation 
• Docker — Containerization 
• Docker Hub — Image Registry (versioned: v1, v2, v3...)
• SonarQube — Static Code Analysis & Quality Gates 
• Trivy — Image Security Scanning 
• GitHub — Source Control + Webhook Trigger

𝗖𝗜/𝗖𝗗 𝗣𝗜𝗣𝗘𝗟𝗜𝗡𝗘 𝗪𝗢𝗥𝗞𝗙𝗟𝗢𝗪
1. Developer pushes code → GitHub webhook triggers instantly
2. Jenkins pipeline initiates the automated build
3. SonarQube scans the code for bugs and security issues
4. Docker image is built from the verified codebase
5. Trivy scans the image for vulnerabilities before deployment
6. Scanned images pushed to Docker Hub with version tags (v1, v2...)
7. Containers deployed using latest images and application is live in production

• Trivy scans every Docker image for vulnerabilities before pushing to the registry ,ensuring only clean and safe images reach deployment.
